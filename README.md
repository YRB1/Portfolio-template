# 💼 Portfolio Template

A modern, responsive personal portfolio website template built using **HTML, CSS, and vanilla JavaScript**, designed to showcase projects, skills, and contact information in a clean and professional style.

## 🚀 Live Demo

👉 [View Live](https://yrb1.github.io/Portfolio-template/)

## 🌟 Features

- 🏠 **Hero Section** – Introduction with your name and title  
- 👨‍💻 **Projects Section** – Showcase your top projects  
- 🧠 **Skills Section** – Highlight your key skills (HTML, CSS, JS)  
- 📩 **Contact Section** – Contact form UI (front-end ready)  
- 📱 **Fully Responsive** – Works on mobile, tablet, and desktop  
- 🎨 **Smooth Animations** – Smooth scrolling and hover effects  

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
